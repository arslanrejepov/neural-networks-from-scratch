# Neural Networks From Scratch

This repository explores how neural networks work by implementing them step-by-step from first principles using **NumPy**.

The goal is to understand the mathematical foundations of neural networks instead of relying only on high-level frameworks.

## Topics Covered

* Single neuron model
* Vectorized forward propagation
* Activation functions (ReLU, Sigmoid, Tanh)
* Loss functions
* Gradient descent
* Backpropagation
* Training a small neural network

## Repository Structure

```
01_single_neuron
02_forward_propagation
03_activation_functions
04_loss_functions
05_gradient_descent
06_backpropagation
07_simple_neural_network
```

## Goals

* Build intuition for how neurons compute outputs
* Understand how gradients update weights
* Implement neural network training from scratch
* Prepare for implementing deep learning models with frameworks later
